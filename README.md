## üé® Customization

### Colors
The color scheme can be easily customized by modifying CSS custom properties in `style.css`:

```css

    --primary-color: #ffffff;
    --secondary-color: #000000;
    --accent-color: #6366f1;
    --glass-bg: rgba(255, 255, 255, 0.1);
    --glass-border: rgba(255, 255, 255, 0.2);
```
